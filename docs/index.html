Team Fortress 2 UI Kit

Hi folks, this is a UI Kit put together by me based roughly on the UI elements of Team Fortress 2.

Using this Kit

You need the teamfortress.css file located in the styles folder. You also need both the fonts located in the fonts folder, then you're good to go!

Layout

I haven't provided much provision for layout, the classes I have provided is the .grid class which will display a grid with two equal columns. If you want more columns use the following additional classes .grid-col3, .grid-col4, .grid-col5, .grid-col6.

You can add a gap between each grid item by appending the class .grid-gap. Finally I've added a padded box class that adds 20px padding to a grid item or any container. .padded-box

Other than those classes, you'll have to improvise and roll your own styles.

Colours

By default the background of the page is set to the dark background you see here. You can change this to the light variant by adding the .background-light class to the body.

Buttons

Buttons are available in two variants, light and dark as well as 3 different sizes. For dark buttons add the .dark-button class.

Tags

Tags are based on the item tags in TF2. Each tag is a span element and has two sizes, the default small and an optional large version, add .tag-large to element. By default the colour of the tag is the common item quality in TF2, but you can specifiy different item quality colours. Inspect these elements to see the class names.

Item Boxes

These boxes are based on the design of the backpack item grid. Each box is made up of a few elements, the box itself which takes on the maximum width of its parent, an inner item image (optional) which has a max width of the box, and finally the item name. The border colour of the box as well as the title colour can be changed by adding a relevant colour class to each.

An optional element (.item-box-effect) can be added inside the item-box to add an on-hover effect. The colour of which is determined by adding a relevant background colour class to the (.item-box-effect) class, like so; (.item-box-effect strange-bg)

Item Detail Boxes

These boxes are based on the item detail or item attribute boxes seen in TF2. They consist of a containing box .item-info-box which expands to the width of it's parent container, which itself contains an item title .item-info-box-title whose colour can be modified using the colour classes above. Below the title is an .item-info element, then finally an unordered ul.item-info-stats list element which contains a list of stats, or attributes.

Each item in this list can be either positive, negative or neutral. Use the following classes to change the colours; .att-positive, .att-negative
