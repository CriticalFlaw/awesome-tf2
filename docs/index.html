awesome-tf2

A curated list of awesome Team Fortress 2 resources, tools and more. To add or update a TF2 resource or site, please submit a pull request. Check the contribution guidelines for more info.

This page was created using the tf2-ui-kit by GingerBunny.

Annual Charity Events

  • Blapature Co. - EU-based charity livestream hosted by a group of TF2 YouTubers.
  • hugs.TF - UK-based charity livestream event benefiting Special Effect.
  • TFConnect - Community ran charity event focused on Mann vs. Machines, benefiting Special Effect.
  • Tip of the Hats - Charity livestream benefiting One Step Camp. Ran from 2013-2021.
